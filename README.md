## Quick start
1. Clone this repo
```
git clone https://github.com/org-not-included/simple_flask_app
cd simple_flask_app
```
2. Run the setup script
```
source setup/run_server.sh
```
3. Go to http://127.0.0.1:4020/ in your web-browser.
4. Stop the server
```
kill -9 $(lsof -ti:4020)
```
